Getting started
=================

* Clone this repo
* `npm install` to install all req'd dependencies (also you need to intstall dependencies in folder 'client')
* `npm run dev` to start development

About Project
================

This project uses:

1. Client:
  * React
  * Redux
  * React-Redux
  * React-Router
  * Bootstrap
  * Redux-Thunk
  
2. Server:
  * Node.js
  * Koa
  * Koa-Passport
  * Mongoose
  * Passport-Jwt
  * Bcryptjs
